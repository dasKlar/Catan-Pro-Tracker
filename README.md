# Catan Board - Pro Score Tracker 🏆

A professional, lightweight Java Swing application designed to track scores, settlements, and special achievements for the board game **Catan**.

## ✨ Features
- **Real-time Scoring:** Automatically calculates points for settlements (1pt) and cities (2pts).
- **Smart Conversion:** Upgrading to a city automatically consumes a settlement.
- **Dynamic Leaderboard:** Tracks "Longest Road" (min 5) and "Largest Army" (min 3) bonuses (+2pts).
- **Turn Management:** Switch between 3-4 players with a dedicated UI color for each player.
- **English UI:** Fully localized for international use.

## 🛠 Tech Stack
- **Language:** Java 17+
- **Library:** Java Swing (Standard GUI)
- **Architecture:** OOP (Object-Oriented Programming)

## 🚀 How to Run
1. Clone the repository: `git clone https://github.com/YOUR_USERNAME/Catan-Pro-Tracker.git`
2. Open the project in **IntelliJ IDEA**.
3. Run the `CatanTracker.java` file.
